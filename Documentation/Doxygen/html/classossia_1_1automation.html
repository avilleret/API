Detailed Description

The automation class.

An automation is a time process that represents a curve of one or multiple parameters.

It drives (i.e. sends messages) to an address, according to a driving value. Its driven net::address_base can be either of numeric type (val_type::INT, val_type::FLOAT...) or of val_type::TUPLE, if the tuple only has numeric elements (e.g. Tuple{Float, Float}).

The driving value can either be a single Behavior or a Tuple of Behavior, in accordance to the type of the driven net::address_base.
