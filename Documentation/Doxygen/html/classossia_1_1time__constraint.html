Detailed Description

The time_constraint class.

#TimeConstraint is use to describe temporal structure precising which amount of time (the duration) are between two #TimeEvents.

The duration can be fixed or between a minimal and a maximal duration.
#TimeConstraint is also a #Clock and a #TimeProcess container.
